# Scrum Facilitator

An interactive web tool for running Scrum ceremonies — Sprint Planning, Daily Stand-up, Sprint Review, and Retrospective — with guided facilitation flows, timers, and educational context.

Part of the [Agile Tools](https://github.com/bthos) suite built on Management 3.0 and ICAgile source materials.

## Stack
React 18 · TypeScript · Vite · Tailwind CSS · react-i18next (EN/RU)

## Development
```bash
npm install
npm run dev
```

## Deploy
GitHub Pages via GitHub Actions on push to `main`.
